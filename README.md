### 简介

- 接入腾讯"兔小巢"实时反馈平台，构建的WEB端社区演示demo
- 用于中小型项目实现收集用户的反馈和基本的社区服务，且无需投入太多开发资源

### 技术选型
#### 后端
- 基础框架：Spring Boot 2.1.18.RELEASE

- 持久层框架：

- 安全框架：

- 数据库连接池：阿里巴巴Druid 1.1.10

- 日志打印：logback

- 其他：hutool 等。

#### 前端
 
- 基础框架：layui 2.5.7

- JavaScript框架：jQuery
